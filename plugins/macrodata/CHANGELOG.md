# @macrodata/opencode

The changelog for the macrodata plugin lives at the **repo root**:
[`../../CHANGELOG.md`](../../CHANGELOG.md).

This package is `ignore`d by changesets (the root package is the versioned one —
see `.changeset/config.json`), so no release entries are written here. This file
exists only because `changesets/action` reads every workspace package's
`CHANGELOG.md` when building the release PR and errors if one is missing.
